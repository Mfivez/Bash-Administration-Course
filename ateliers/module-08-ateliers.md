# Atelier module 8 — Projet final `admin_toolkit.sh`

## Objectif

Assembler les scripts créés dans un menu d'administration.

## Menu attendu

```text
1. Afficher les informations système
2. Vérifier l'espace disque
3. Lister les utilisateurs
4. Préparer la création d'utilisateurs depuis un fichier
5. Analyser les logs
6. Sauvegarder les données
7. Générer un rapport complet
8. Quitter
```

## Exigences

- utiliser des fonctions ;
- valider les choix ;
- produire des rapports ;
- réutiliser les scripts existants ;
- gérer les erreurs ;
- documenter le script.

## Démonstration

Préparez une démonstration courte et structurée :

1. affichage du menu ;
2. diagnostic serveur ;
3. analyse de logs ;
4. sauvegarde ;
5. rapport complet.

## Bonus

Ajouter un mode non interactif :

```bash
./admin_toolkit.sh report
./admin_toolkit.sh logs
```
