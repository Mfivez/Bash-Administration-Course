# Évaluation

## Évaluation continue

Chaque jour, l'apprenant produit un livrable. Le formateur vérifie :

- exécution correcte ;
- compréhension des commandes ;
- capacité à expliquer les erreurs ;
- propreté minimale du script ;
- suivi individuel renseigné.

## Mini-défis quotidiens

| Jour | Défi |
|---:|---|
| 1 | Recréer l'arborescence NovaCorp sans aide |
| 2 | Corriger un problème de permission et rediriger les erreurs |
| 3 | Écrire un script qui valide un fichier passé en argument |
| 4 | Lire un fichier ligne par ligne et générer un rapport |
| 5 | Extraire les erreurs d'un log et les compter |
| 6 | Produire un diagnostic serveur |
| 7 | Créer une sauvegarde horodatée avec log |
| 8 | Présenter la boîte à outils complète |
